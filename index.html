Introduction

Skywarp++ is an (BSD license) open source server library that enables the use of remote procedure call for exposition of functionalities and real time data streaming capabilities to any application written in c++.

The c++ applications that uses skywarp++, can expose your functionalities through of JSON-RPC 2.0 subprotocol in order to be used by clients that supports websocket transport protocol.

Furthermore, skywarp++, enable application to implement real time data streaming to your client in a push model, through of a built-in publish-subscriber pattern.

Project Status

Main features are in testing phase. New features are still in development.

Main features

  • JSON-RPC 2.0 compatible, exposition of application functionalities. (Batch method call is not still supported).
  • Real time data streaming support with built-in publish-subscribe pattern.
  • Built-in client session control.
  • Asynchronous messaging model.
  • Message format (Text in Json-RPC format)
  • Websocket full support: RFC6455.
  • TLS support (through websocket++)
  • Ipv6 support (through websocket++)

Possible applications:

  • Telemetry systems.
  • Real time monitoring systems.
  • Real time market data streaming.
  • Data adquisition and SCADA applications.

For more information and examples please visit the project site.
